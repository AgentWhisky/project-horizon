Setup Heroku Docker Container
-----------------------------
heroku login # Make sure to install heroku cli first
docker ps  # Make sure to have docker installed and running locally
heroku container:login  
heroku stack:set container --app heroku-app-name

Deploying on Heroku
---------------------
Check heroku-deploy github readme for useful options
  > https://github.com/AkhileshNS/heroku-deploy


Generate 128-character hex string for AUTH
--------------------
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"


Generate file structure tree
--------------------
PS: tree /F > client-project-structure.txt


GIT Commands
--------------------
List changes in last month
  > git log --since="1 month ago" --name-only --pretty=format: | sort -u >> changes.txt

List changes since date
  > git log --since="2025-10-01" --name-only --pretty=format: | sort -u >> changes.txt
